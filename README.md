# Sistema de Agendamento de Salas em C

Este sistema permite, através de um menu interativo de opções, gerenciar informações sobre as salas (para administradores), além de agendar e cancelar reservas, e exibir lista de salas e reservas cadastradas.
Para o desenvolvimento do sistema, foram utilizados inúmeros recursos da linguagem C, como bibliotecas, constantes, variáveis, ponteiros, estruturas heterogêneas de dados, funções, laços de repetição, condições, condições ternárias, alocação dinâmica de memória, leitura e escrita em arquivos, dentre outros recursos. Os arquivos “salas.txt”, “reservas.txt” e “cancelamentos.txt” servem como forma de armazenamento de informações sobre as salas, reservas e cancelamentos, respectivamente.

As estruturas e seus respectivos campos presentes no sistema são:
* Data: três variáveis do tipo inteiro para o dia, mês e ano.
* Horário: duas variáveis do tipo inteiro para as horas e minutos.
* Sala: duas variáveis do tipo inteiro para id e lotação máxima, e duas cadeias de caracteres para o nome e a descrição.
* Reserva: três variáveis do tipo inteiro para id, id da sala e quantidade de pessoas, uma estrutura do tipo Data e duas estruturas do tipo Horário, uma para o horário de início e outra para o horário de término da reserva.

As funções presentes no sistema são:
* void obterDataEHoraAtual(Data *dataAtual, Horario *horarioAtual): Seus parâmetros são um ponteiro para uma estrutura do tipo Data, e um ponteiro para uma estrutura do tipo Horário. Essa função é responsável por obter a data e o horário do momento em que o arquivo executável é aberto. Não retorna nada.
* Sala * lerSalas(FILE *arq, char *nomeArq, int *qtdSalas): Seus parâmetros são um ponteiro para uma estrutura do tipo FILE, uma cadeia de caracteres e um ponteiro para uma variável do tipo inteiro. Essa função é responsável por ler todas as informações das salas cadastradas no arquivo “salas.txt”, incluindo id, nome, descrição, e lotação máxima de cada sala. Retorna um vetor de estruturas do tipo Sala.
* Reserva * lerReservas(FILE *arq, char *nomeArq, int *qtdReservas, Data dataAtual, Horario horarioAtual): Seus parâmetros são um ponteiro para uma estrutura do tipo FILE, uma cadeira de caracteres, um ponteiro para uma variável do tipo inteiro, uma estrutura do tipo Data e uma estrutura do tipo Horário. É responsável por ler todas as informações das reservas cadastradas no arquivo “reservas.txt”, incluindo id, id da sala, data, horário de início, horário de término e quantidade de pessoas. Não lê informações das reservas que sejam anteriores à data e horário atual. Retorna um vetor de estruturas do tipo Reserva.
* void listarSalas(Sala *salas, int qtdSalas): Seus parâmetros são um vetor de estruturas do tipo Sala e uma variável do tipo inteiro. É responsável por exibir as informações de todas as salas cadastradas. Não retorna nada.
* void listarReservas(Reserva *reservas, int qtdReservas, Sala *salas, int qtdSalas): Seus parâmetros são um vetor de estruturas do tipo Reserva, duas variáveis do tipo inteiro e um vetor de estruturas do tipo Sala. É responsável por exibir as informações de todas as reservas cadastradas. Não retorna nada.
* void listarReservasDeUmaSala(Reserva *reservas, int qtdReservas, Sala *salas, int qtdSalas): Seus parâmetros são um vetor de estruturas do tipo Reserva, duas variáveis do tipo inteiro e um vetor de estruturas do tipo Sala. É responsável por exibir as informações de todas as reservas de uma sala informada pelo usuário. Não retorna nada.
* Reserva * agendarSala(Reserva *reservas, int *qtdReservas, Sala *salas, int qtdSalas, Data dataAtual, Horario horarioAtual): Seus parâmetros são um vetor de estruturas do tipo Reserva, um ponteiro para uma variável do tipo inteiro, um vetor de estruturas do tipo Sala, uma variável do tipo inteiro, uma estrutura do tipo Data e uma estrutura do tipo Horário. É responsável por realizar o agendamento de uma sala, ou seja, o cadastro de uma reserva. No entanto, antes de reservar, verifica-se se a sala desejada tem capacidade para a quantidade de pessoas da reserva, além de recomendar uma sala com capacidade mais ideal para a quantidade de pessoas, caso haja. Verifica-se também se é possível reservar para o dia e horário desejado, ou se já existe uma reserva para esse período. Retorna um vetor de estruturas do tipo Reserva.
* Reserva * cancelarReserva(Reserva *reservas, int *qtdReservas, Reserva *listaAnteriorDeCancelamentos, int *qtdCancelamentos): Seus parâmetros são dois vetores de estruturas do tipo Reserva e dois ponteiros para variáveis do tipo inteiro. É responsável por cancelar uma reserva informada pelo usuário. Retorna um vetor de estruturas do tipo Reserva.
* Sala * cadastrarSala(Sala *salas, int *qtdSalas): Seus parâmetros são um vetor de estruturas do tipo Sala e um ponteiro para uma variável do tipo inteiro. É responsável por realizar o cadastro de uma nova sala, no entanto, somente se o usuário informar a senha de administrador corretamente. Retorna um vetor de estruturas do tipo Sala.
* Sala * excluirSala(Sala *salas, int *qtdSalas, Reserva **reservas, int *qtdReservas): Seus parâmetros são um vetor de estruturas do tipo Sala, dois ponteiros para variáveis do tipo inteiro e um ponteiro para um vetor de estruturas do tipo Reserva. É responsável por excluir uma sala, no entanto, somente se o usuário informar a senha de administrador corretamente. Realiza também a exclusão das reservas cadastradas para a sala excluída. Retorna um vetor de estruturas do tipo Sala.
* void passarSalasParaArquivo(FILE *arq, char *nomeArq, Sala *salas, int qtdSalas): Seus parâmetros são um ponteiro para uma estrutura do tipo FILE, uma cadeia de caracteres, um vetor de estruturas do tipo Sala e uma variável do tipo inteiro. É responsável por escrever as informações de todas as salas cadastradas no arquivo “salas.txt”. Não retorna nada.
* void passarReservasParaArquivo(FILE *arq, char *nomeArq, Reserva *reservas, int qtdReservas): Seus parâmetros são um ponteiro para uma estrutura do tipo FILE, uma cadeia de caracteres, um vetor de estruturas do tipo Reserva e uma variável do tipo inteiro. É responsável por escrever as informações de todas as reservas cadastradas no arquivo “reservas.txt”. Não retorna nada.
* void passarCancelamentosParaArquivo(FILE *arq, char *nomeArq, Reserva *cancelamentos, int qtdCancelamentos): Seus parâmetros são um ponteiro para uma estrutura do tipo FILE, uma cadeia de caracteres, um vetor de estruturas do tipo Reserva e uma variável do tipo inteiro. É responsável por adicionar as informações de todas as reservas canceladas no arquivo “cancelamentos.txt”. Não retorna nada.
* void liberarMemoria(Sala **salas, int qtdSalas, Reserva **reservas, Reserva **cancelamentos): Seus parâmetros são um ponteiro para um vetor de estruturas do tipo Sala, uma variável do tipo inteiro e dois ponteiros para vetores de estruturas do tipo Reserva. É responsável por liberar toda a memória alocada dinamicamente durante a execução do programa. Não retorna nada.
